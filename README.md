## Overview

**UAF-Repo** is a tool for converting system models that use the Unified Architectural Framework (UAF), defined by the Object Management Group (OMG), into a graph-based format for use in Neo4j.

The repository includes:
- Resources for setting up Neo4j using Docker containers
- Integration components for extending functionality with a Claude MCP server
